# apartment-manager

Apartment Management Web App built with React, Vite, Tailwind CSS, and Firebase.

## Features

- Create/Join apartments with room codes
- Admin dashboard for managing residents, bills, and announcements
- Resident dashboard for viewing bills and messages
- Real-time data sync via Firebase
- Payment settings with QR code support
- AI Insights for bill management

## Setup

```bash
npm install
npm run dev
```

## Firebase Setup

1. Create a Firebase project
2. Enable Firestore and Authentication (anonymous)
3. Copy config to src/firebase.js
